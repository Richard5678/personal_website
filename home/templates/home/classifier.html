Outline

This project classifies creatures based on their structure which is given in a form of undirected and unweighted graph.

Essentially, there are a set of commands that lets the user creates a node or connect, disconnect a node from another node. In addition, there are also commands that ask the program to classify the current structure or delete the whole structure.
